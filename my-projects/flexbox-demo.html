Flexbox Demo

Interactive demonstration of Flexbox layout fundamentals

About this project

This project demonstrates the power of CSS Flexbox for building responsive layouts with alignment, spacing, and ordering features.

Interactive Demo

Resize the browser window to see flex items adjust and use hover effects to see subtle animations.

Features

  • Flex container properties (justify-content, align-items, flex-wrap)
  • Responsive design with flex items
  • Hover effects for interactivity

Skills Demonstrated

HTML, CSS Flexbox, responsive design, interactivity with CSS and minimal JS
